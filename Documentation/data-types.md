# Data Types
There are four main data types: int, float, bool, and string.

## Ints
Short for integer, they store a single, whole number, negative or positive.
They can be declared as follows:
```
let x: int = 5;
```

Note: If they are delcared with a decimal (e.g. 5.6), it will be ignored completely and cut off.

## Floats
Floats, similar to ints, store numbers. But rather than a whole number, they store floating-point numbers. This means they can contain decimals.
```
let y: float = 4.5;
```

## Strings
Strings store text. They are declared with speech-marks surrounding them ("Hello").
```
let str: string = "Hello!";
```

## Bools
Bool store either true or false.
```
let on: bool = true;
```
